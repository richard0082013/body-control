# Body Control — 家庭训练进度面板

该项目基于 Vue 3 与 Vite 搭建，用于追踪一周循环的居家训练与健康记录。配合 `../docs/workout-plan.md` 与 `../docs/nutrition-plan.md` 中的方案，可在网页上勾选每日训练项、查看改写的饮食建议，并记录身体反馈。

## 开发命令

```bash
# 安装依赖
npm install

# 本地开发
npm run dev

# 生产构建
npm run build
```

## 主要功能
- 根据用户填写的身高体重、血糖/血压状况与运动习惯动态生成 7 天训练与饮食组合建议。
- 每日训练项可勾选完成情况，自动计算周度完成度并以进度条呈现。
- 支持记录当日感受、血糖/血压观测情况，数据存储于浏览器 `localStorage`。
- 自动突出显示当天计划，便于快速查看。
- 动作卡片内提供简易动画示意、步骤与安全提示，帮助正确理解动作细节。
- 饮食页面同步展示三餐/加餐建议与注意事项，可结合文档进行个性化调整。

## 目录结构概览

```
web/
├── src/
│   ├── components/         # UI 组件（周度总览、每日卡片、饮食面板等）
│   ├── composables/        # 自定义逻辑（localStorage 进度 & 体况信息）
│   ├── data/               # 训练与饮食计划数据及生成器
│   ├── App.vue             # 页面布局
│   └── style.css           # 全局样式与设计变量
└── docs/                   # 训练与饮食文档（位于仓库根目录）
```

如需扩展，可考虑整合血糖记录表单、提醒通知或图表展示功能。
