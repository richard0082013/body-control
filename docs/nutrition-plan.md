# 一周控糖控压饮食建议

> 适用人群：2 型糖尿病、血压偏高的上班族程序员（身高 177cm，体重 109kg），需要控制总热量、稳定血糖、兼顾脑力与体力需求。以下方案以每日 3 餐 + 1-2 次小加餐为基础，可根据血糖记录和体重波动微调。

## 共通原则
- **碳水分配**：将每日碳水均匀分布到各餐，占总能量约 45%-50%，优先选择低 GI 全谷物（燕麦、糙米、全麦面包、杂豆）。
- **蛋白质**：每餐至少含 1-2 份优质蛋白（瘦肉、鱼虾、豆制品、鸡蛋、低脂乳制品），总量约 1.2g/kg 体重（130g 蛋白质左右）。
- **脂肪**：使用橄榄油、亚麻籽油等健康脂肪，控制烹饪油 ≤25g/日，避免反式脂肪。
- **蔬菜**：每餐搭配 2-3 种彩色蔬菜，保证每日 500-700g。
- **水果**：血糖稳定时可在非空腹时摄入低 GI 水果（例如：半个苹果、猕猴桃 1 个），每日不超过 200g，纳入碳水计算。
- **喝水**：每日 2000ml 左右白开水或无糖茶；限制含糖饮料与高盐汤品。
- **盐摄入**：≤5g/日，注意调味品中隐形盐（酱油、咸菜）。
- **补剂建议**：在医生同意下，可考虑补充多维与鱼油帮助心血管代谢。

## 每日模板（周一至周七）

| 餐次 | 食材举例 | 营养要点 |
| --- | --- | --- |
| 早餐 | - 黑麦或燕麦片 40-50g + 无糖豆浆 250ml<br>- 水煮蛋 1 个或鸡胸肉 50g<br>- 时蔬（生菜、番茄、黄瓜）一碗 | 复合碳水 + 蛋白质，延缓血糖波动；蔬菜提升饱腹感 |
| 上午加餐（可选） | - 无盐坚果 10g + 半个苹果<br>或<br>- 原味低脂酸奶 150ml | 控制在 150 kcal 内，防止低血糖 |
| 午餐 | - 糙米/杂粮饭 100g（生重）<br>- 清蒸鱼或瘦肉 120g<br>- 炒西兰花胡萝卜 200g（少油）<br>- 海带豆腐汤（少盐） | 主食占 1/4，蛋白 1/4，蔬菜 1/2；海带帮助补碘、钾 |
| 下午加餐（可选） | - 一小杯无糖豆浆或绿茶<br>- 礼貌量的黄瓜条、樱桃番茄 | 提供水分和微量营养素，控制饥饿感 |
| 晚餐 | - 荞麦面或藜麦 80g（干重）<br>- 蒸鸡腿去皮或豆腐煲 100-120g<br>- 炒蘑菇、菠菜 200g<br>- 一小碗紫菜虾皮汤 | 晚餐碳水略减，增加高纤维蔬菜和优质蛋白 |

## 每周变化建议

### 周一：基础热量 1800 kcal 左右
- 主食：糙米饭 + 燕麦粥
- 蛋白：清蒸鳕鱼、鸡胸肉
- 蔬菜：西兰花、胡萝卜、黄瓜
- 调味：蒜蓉、柠檬汁、少量橄榄油

### 周二：提高膳食纤维
- 主食：黑麦面包 2 片 + 杂豆饭
- 蛋白：卤牛腱 80g + 豆腐 100g
- 蔬菜：甘蓝、秋葵、番茄
- 特别添加：奇亚籽 10g 混入酸奶，增强饱腹

### 周三：轻断碳水日（保持稳定）
- 早餐：全麦卷饼夹鸡蛋 & 生菜
- 午餐：藜麦鸡肉沙拉（橄榄油 5g）
- 晚餐：番茄豆腐鲜虾汤 + 蒸南瓜 120g
- 减少主食份量，搭配更多蔬菜防止血糖波动

### 周四：补充优质脂肪
- 主食：糙米 + 荞麦饭
- 蛋白：红烧鲑鱼 120g
- 蔬菜：芦笋、菠菜
- 小加餐：牛油果 1/4 个 + 坚果 10g

### 周五：脑力支持日（适合高工作强度）
- 早餐：燕麦片加无糖豆奶，撒亚麻籽粉
- 午餐：三文鱼饭盒（主食糙米 80g，搭配紫甘蓝）
- 晚餐：罗宋汤 + 烤鸡腿去皮 100g + 西葫芦
- 加餐：蓝莓一小把（控制在 80g 内）

### 周六：训练量较高，可适度增加蛋白
- 主食：杂粮饭 + 全麦意面（晚餐 60g）
- 蛋白：火鸡胸肉 100g、虾仁 100g
- 蔬菜：甜椒、菜花、豌豆
- 加餐：蛋白粉 1 勺（选择低糖配方，如需）

### 周日：轻松恢复日
- 主食：小米粥 1 碗 + 玉米半根
- 蛋白：清蒸豆腐 150g + 鸡蛋羹
- 蔬菜：莲藕、木耳、黄瓜凉拌
- 可搭配红枣姜茶（无糖），促进血液循环

## 血糖与血压监测配合
- 记录每餐前后血糖（空腹、餐后 2 小时），比较不同主食的影响。
- 若餐后血糖 >10 mmol/L，优先减少下一餐精制碳水、增加蔬菜和蛋白比例。
- 血压监测至少每日 2 次，注意高钠食品（腌制品、加工肉）的摄入频率。
- 若训练日当天出现低血糖征兆（手抖、出汗），及时补充 15g 快速碳水（葡萄糖片或半杯果汁），15 分钟后复测。

## 调整策略
- **体重下降平台**：在医生认可下，可采用 16:8 轻断食模式（早餐 9 点，晚餐 17 点），但务必监测血糖。
- **血糖偏高时**：减少水果频次，选择全麦杂粮替换部分米饭；关注睡眠质量。
- **血压偏高时**：加重钾镁摄入（菠菜、香蕉、豆类），减少咖啡因摄入量，增加冥想时间。

> 📌 建议每 2 周与营养师或医生复盘饮食与血糖记录，适时微调总能量摄入与运动强度。
